# Perfect Squares

[Leetcode Link](https://leetcode.com/problems/perfect-squares/)

## Problem:

Given a positive integer _n_, find the least number of perfect square numbers (for example, `1, 4, 9, 16, ...`) which sum to _n_.

## Example:

```
Input: n = 12
Output: 3
Explanation: 12 = 4 + 4 + 4.
```

```
Input: n = 13
Output: 2
Explanation: 13 = 4 + 9.
```
